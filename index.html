DEFINITIONS

1. What is the full meaning of HTML?
It simply means "Hyper Text Mark-up Language"
2. Define HTML boiler plate.
HTML boiler plate is the basic structure,essential HTML code that is required to create a web page. It is generated using the "!" on the "index.html" file.
3. Differentiate between the Ordered and Unordered Lists.
Ordered list is a list of items that is formatted with a number. In HTML, It is created using the "ol" element.
While the Unordered list on the other hand is a list of items that are formatted with bullet points indicating each item. In HTML, It is created using the "ul" element.
